###  `AE工具栏及快捷键使用`:

> ```html
> AE 快捷键:
> 
> 
> 
> 	UU 展开/关闭 fx
> 
> 
> 
> 	shift + F9 关键帧添加缓入
> 
> 	ctrl + shift + F9 关键帧添加缓出
> 
> 
> 	帧速率: 25 帧
> 
> 	
> 
> 	ctrl + alt + f 适配窗口大小
> 
> 
> 
> 	ctrl +  alt + shift + h  适配宽度
> 
> 
> 
> 	ctrl +  alt + shift + G 适配高度
> 
> 
> 
> s 缩放
> 
> 
> 
> alt + [  ]   剪裁时间轴
> 
> 
> 
> 
> ctrl + y 新建纯色图层
> 
> 
> ```
>
> 

> `工具顶栏带有小三角按住左键不松手`
