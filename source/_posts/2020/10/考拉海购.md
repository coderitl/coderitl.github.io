###  滤镜效果(高斯模糊):

```scss
对两个 div 进行层叠,并实现高斯模糊

 .box {
     margin-top: 200px;
     width: 100%;
     height: 50px;
     position: absolute;
     top: 0;
     left: 0;
     right: 0;
        }

.blue {
    position: absolute;
    background-color: blue;
    left: 0;
    right: 0;
    bottom: 0;
    top: 0;
    /* filter: blur(8px); */
}


+ 层级关系

	.box>.blue

```
