JavaScript-三种动态创建元素的区别

发表于2020-11-29|更新于2020-11-30|javascript
|阅读量:

三种动态创建元素的区别

  • document.write()
  • element.innerHTML
  • document.createElement()

区别

  1. docuemnt.write是直接将内容写入特定的内容流,但是文档流执行完毕,则它会导致页面全部重绘

  2. innerHTML 是将内容写入某个DOM节点,不会导致页面全部重绘

  3. innerHTML创建多个元素效率更高(不要拼接字符串,采取数组形式拼接)

  4. createElement()创建多个元素效率稍微低一点,但是接轨更加清晰

采取数组时,innerHTML效率要比creatElement高
