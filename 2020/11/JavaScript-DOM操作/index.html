    element.getAttribute()

  • 获取属性值

    • element.属性 获取属性值
    • element.getAttribute('属性')

  • 区别

    • element.属性 获取内置属性值(元素本身自带的属性)

    • elemetn.getAttribute('属性')主要获得自定义的属性

  • 设置属性值

    • element.属性='值'
    • element.setAttribute('属性','值')

  • 特殊设置类

    元素.setAttribute('class','新值') // class 特殊 写的时 class 不是 className
