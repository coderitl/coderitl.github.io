  • 右击 点击 git bash here

Github配置：

  • 进入github主页注册或登录

    Github 请点击

首先在本地创建ssh key:

输入邮箱后按下回车,会在下图蓝框位置出现配置文件

进入该目录下(C:\Users\你的电脑用户名\.ssh), 打开id_rsa.pub，复制里面的key

  • 返回 github,完成如下操作

  • 验证是否成功

  • 接下来我们要做的就是把本地仓库传到github上去，在此之前还需要设置username和email，因为github每次commit都会记录他们。

  • 新建仓库

  • 命令粗略解释

  • 选择带有index.html文件的进行点击,没有则不配置

  • 下拉选择Github Pages

  • 点击Save

  • 刷新再次进入该页面即可配置静态网页资源
