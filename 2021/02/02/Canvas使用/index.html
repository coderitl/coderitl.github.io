圆弧绘制

  • 参数解释

    1. 100 canvas内坐标起始点
    2. 50 圆的半径
    3. 0 绘制起点
    4. 2*Math.PI 圆圈弧度
    5. true 逆时针

  • 效果图

0,1指的是开始和结束的位置,单位如果是数字,则代表一个圆弧的弧度(一个圆的弧度是Math.PI * 2 ,约等于 7 个弧度) 所以如果为两个参数的差为 7 ，并且是 顺时针方向,则代表绘制一个圆
