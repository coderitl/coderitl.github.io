Python-Re在线练习

  • 在线网站: 前往练习

  • 规则

基础练习

或 ：

  • 练习一

    Task Text
    Match abc123xyz
    Match define”123”
    Match var g = 123

    个人: [a-z]+ | \s [\d]+ | ["123"]

  • 练习二

    Task Text
    Match cat.
    Match 896.
    Match ?=+.
    Skip abc1

    答案: ...\.

  • 练习三

    Task Text
    Match can
    Match man
    Match fan
    Skip dan
    Skip ran
    Skip pan

    理解: Skip是排除在外的

    个人答案: [cmf]an

  • 练习三

    Task Text
    Match hog
    Match dog
    Skip bog

    个人答案: [hd]og

非:

  • 练习一

    Task Text
    Match Ana
    Match Bob
    Match Cpc
    Skip aax
    Skip bby
    Skip ccz

    个人答案: [^(a-c)]

  • 练习二

    Task Text
    Match aaaabcc
    Match aabbbbc
    Match aacc
    Skip a

    个人答案: a*b*c
